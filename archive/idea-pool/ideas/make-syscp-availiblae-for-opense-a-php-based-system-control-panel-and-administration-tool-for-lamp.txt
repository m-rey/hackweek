====== make syscp for opensuse and sles available ======

Adaption of the debian packages to opensuse 11.0, factory and sles 10.
Adaption of ftp (move from proftp to pureftp)
Rewriting and actualizations of Manual in syscp package.
Rewriting and actualizations of Manual on project syscp 
https://wiki.syscp.org/contrib/installation/opensuse/11.0

===== Description =====
/* Describe your idea here. */
syscp is an a php based System Control Panel and administration tool for lamp servers which host more than one webdomain/Website.

It is an tool easy to handle the administration of websites domains, subdomains email addresses, mysql date bases, phpmyadmin, webstats (webalizer), ftp accounts, password protections for directories, DNS, webmail, even mail filter configuration and it is under GPL. It supports some language preference (german, english, french, catalan, chinese, spanish, portuguese). 

available using opensuse build service for opensuse 10.2, 10.3, 11.0 factory and sles10
http://software.opensuse.org/search?baseproject=ALL&p=1&q=syscp
